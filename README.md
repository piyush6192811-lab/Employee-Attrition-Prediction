# Employee-Attrition-Prediction
Employee Attrition Prediction using Machine Learning with Python and Random Forest Classifier.

## Overview
This project predicts whether an employee is likely to leave a company using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Algorithm
Random Forest Classifier

## Dataset
IBM HR Analytics Employee Attrition Dataset

## Accuracy
86.73%
