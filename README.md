# Employee Attrition Prediction Using Machine Learning

## 📌 Project Overview
Employee attrition is one of the major challenges faced by organizations. This project uses Machine Learning to predict whether an employee is likely to leave the company based on various HR-related factors such as age, job role, monthly income, overtime, years at the company, and other employee attributes.

The model is developed using the IBM HR Analytics Employee Attrition dataset and the Random Forest Classifier algorithm.

---

## 🎯 Objectives
- Predict employee attrition using Machine Learning.
- Perform data preprocessing and feature engineering.
- Analyze employee data using Exploratory Data Analysis (EDA).
- Train and evaluate a classification model.
- Help organizations identify employees who are at risk of leaving.
